# 🎬 Movie Audience Analyzer

A Streamlit dashboard for analyzing and ranking moviegoer data by audience count.

---

## 📋 Description

Movie Audience Analyzer is an interactive web dashboard built with Streamlit.
It lets you explore movie audience data, visualize trends, and rank films by the number of viewers.

---

## ✨ Features

- 📊 Visualize moviegoer counts with interactive charts
- 🏆 Rank movies by total audience size
- 🔍 Filter and search by title, genre, or date
- 📅 Analyze trends over time
- 💾 Export sorted results as CSV

---

## 🚀 How to Run

**1. Clone the repository**

`git clone https://github.com/your-username/movie-audience-analyzer.git`

**2. Install dependencies**

`pip install -r requirements.txt`

**3. Run the Streamlit app**

`streamlit run app.py`

**4. Open in browser**

`http://localhost:8501`
